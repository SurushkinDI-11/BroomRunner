# BroomRunner

**Broom Runner** — компьютерная аркадная игра в жанре «раннер» (бесконечный бег с препятствиями).

## Описание

Главным действующим лицом является работник коммунальной сферы — **дворник**, осуществляющий уборку городских территорий в ночное время.

**Цель игры:** преодоление максимально возможного расстояния без столкновения с препятствиями, набор очков и улучшение личного рекорда.

---

## Режим игры

### 🎮 Аркада

Игровой процесс реализован в виде непрерывного автоматического движения главного героя вперёд по прямолинейной трассе, символизирующей городскую улицу. Трасса генерируется **процедурно**, что обеспечивает уникальность каждого забега.

---

## Усложнение игрового процесса

С увеличением времени забега:

- ⬆️ возрастает **базовая скорость** движения героя
- ⬆️ увеличивается **частота появления** препятствий
- ⬆️ растёт **сложность комбинаций** препятствий

---

## Условия завершения игры

> ⚠️ Игра завершается при **любом столкновении** героя с препятствием.

После этого отображается итоговое количество набранных очков и предлагается начать новый забег.

## Установка и запуск

### 💻 Windows (EXE Installer)

1. Скачайте установщик `BroomRunnerInstaller.exe` из [раздела Releases](../../releases)
2. Запустите файл двойным щелчком мыши
3. Следуйте инструкциям мастера установки
4. После завершения установки запустите игру с ярлыка на рабочем столе или из меню «Пуск»

> **Требования:** Windows 7/8/10/11 (64-bit)
